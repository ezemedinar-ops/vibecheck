PRD VibeCheck - Documento Tecnico Completo

1. Introduccion y Vision
Este documento detalla la arquitectura de datos para la aplicacion VibeCheck. El objetivo es utilizar la API de Spotify
para generar graficos de ADN musical y metrics personalizadas para el usuario.
